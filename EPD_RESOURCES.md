# Free Online Chess EPD Resources

This file contains a curated list of free, reputable Extended Position Description (EPD) files for chess engine testing and tuning.

## Comprehensive Collections

### [ChrisWhittington/Chess-EPDs](https://github.com/ChrisWhittington/Chess-EPDs)
A GitHub repository collecting various public domain test suites.
*   **Repo URL**: `https://github.com/ChrisWhittington/Chess-EPDs`

## Specific Test Suites (Direct Downloads)

### Tactical & Positional Tests

*   **Bratko-Kopec Test** (24 Positions)
    *   *Description*: Standard tactical test suite from the 1980s.
    *   **Direct Link**: [bratko-kopec.epd](https://raw.githubusercontent.com/ChrisWhittington/Chess-EPDs/master/bratko-kopec.epd)

*   **Kaufman Test**
    *   *Description*: Famous test positions credited to Larry Kaufman.
    *   **Direct Link**: [kaufman.epd](https://raw.githubusercontent.com/ChrisWhittington/Chess-EPDs/master/kaufman.epd)

*   **LCT2**
    *   *Description*: Famous LCT2 test positions.
    *   **Direct Link**: [lct2.epd](https://raw.githubusercontent.com/ChrisWhittington/Chess-EPDs/master/lct2.epd)

*   **Strategic Test Suite (STS)**
    *   *Description*: Merged file containing STS 1-15. Designed to evaluate long-term understanding of strategical concepts.
    *   **Direct Link**: [STS1-STS15_LAN_v3.epd](https://raw.githubusercontent.com/fsmosca/STS-Rating/master/STS1-STS15_LAN_v3.epd)
    *   *Source*: [fsmosca/STS-Rating](https://github.com/fsmosca/STS-Rating)

*   **Silent But Deadly**
    *   *Description*: Quiet positions designed by Dann Corbitt.
    *   **Direct Link**: [silent-but-deadly.epd](https://raw.githubusercontent.com/ChrisWhittington/Chess-EPDs/master/silent-but-deadly.epd)

*   **Nolot**
    *   *Description*: Famous Nolot test positions.
    *   **Direct Link**: [nolot.epd](https://raw.githubusercontent.com/ChrisWhittington/Chess-EPDs/master/nolot.epd)

*   **Arasan Test Suite & WAC**
    *   *Description*: A collection of test suites used by the Arasan chess engine, including "Win At Chess" (WAC), ECMGCP, IQ4, and others.
    *   **Direct Link (Zip)**: [tests.zip](http://www.arasanchess.org/tests.zip)
    *   *Source*: [arasanchess.org](https://www.arasanchess.org/tests.shtml)

### Move Generation & Integrity (Perft)

*   **Perft (Standard)**
    *   *Description*: Perft stress testers designed to catch special move bugs (en passant, castling, promotions).
    *   **Direct Link**: [perft.epd](https://raw.githubusercontent.com/ChrisWhittington/Chess-EPDs/master/perft.epd)

*   **Perft (Marcel)**
    *   *Description*: 6000+ positions credited to Marcel van Kervinck.
    *   **Direct Link**: [perft-marcel.epd](https://raw.githubusercontent.com/ChrisWhittington/Chess-EPDs/master/perft-marcel.epd)

*   **Perft (Double Checks)**
    *   *Description*: Positions where the side to move is in double check.
    *   **Direct Link**: [perft-ferdy-double-checks.epd](https://raw.githubusercontent.com/ChrisWhittington/Chess-EPDs/master/perft-ferdy-double-checks.epd)
